# goodsAndThatAll

Приложение - аналог корзины в онлайн-магазине Wildberies.

Реализовано:
  - работа аккордиона
  - работа инпутов
  - подсказки
  - динамический подсчет стоимости и колличества товаров, скидок
  - функционал изменения способа доствка и оплаты
  - валидация полей формы оформления заказа
  - адаптивный дизайн
